# yuki utils

A PvP utility mod for Minecraft 1.21.11 Fabric.

## Features

- Armor HUD
- Effect HUD
- Reach Display
- Shulker Tooltips
- Totem Tweaks
- Block Outline
- Custom Tooltips
- Fullbright
- Item Animations
- Nametag Tweaks
- Scoreboard Tweaks
- Chat Tweaks
- HUD Editor
- Click GUI

## Requirements

- Minecraft 1.21.11
- Fabric Loader
- Fabric API
